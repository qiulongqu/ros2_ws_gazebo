# layers in ROS2(完整机械臂仿真工程项目的工作流程)

## 1. 构建“肉身”（机器人建模）

    包含:三维模型 (3D Meshes)和URDF (统一机器人描述格式)文件。

    其中：三维模型文件可以是.stl,.obj,.dae等格式，URDF文件则是描述机器人各个部件的位置和姿态的描述文件。

## 2. 搭建“物理世界”（选择仿真器）

    选择一个开源的仿真器，如Gazebo、V-REP、NVIDIA Isaac Sim、MuJoCo等。

## 3. 打通“神经系统”（底层控制与通信）

    底层控制系统负责控制机器人的动作，如电机驱动、传感器数据采集、底盘控制等。通信系统则负责与其他机器人、环境进行通信，如ROS、Ethernet、CAN等。

## 4. 编写“控制算法”（算法实现）

    算法实现包括机器人动作控制、路径规划、避障、运动学控制等。这里包含机器人正逆运动学。

    重点是：MoveIt 2框架

## 5. 监控与交互（数据可视化）

    数据可视化包括机器人状态监控、轨迹跟踪、机器人行为可视化等。

    重点是：RViz、Webots、ROS2 Dashboard等。
